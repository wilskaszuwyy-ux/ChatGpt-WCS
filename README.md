# Dashboard de valorizacion de obra - abril 2026

Dashboard offline basado en el informe mensual de supervision y valorizacion de la IOARR con CUI 2713723.

## Ejecucion sin internet

Abra `index.html` directamente en el navegador o ejecute:

```bash
npm start
```

Luego visite `http://localhost:4173`.

No usa CDN ni dependencias externas. El PDF fuente esta embebido en `src/pdfData.js` para consulta local desde el panel **Documento fuente**.

## Verificacion

```bash
npm test
```
