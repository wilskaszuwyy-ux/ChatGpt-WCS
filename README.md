# Calculadora de capacidad portante (Terzaghi)

Aplicación en un **solo archivo HTML** (`index.html`) para calcular capacidad portante por Terzaghi.

## Verla en esta plataforma (preview por puerto)

1. Ejecuta:
   ```bash
   python3 server.py
   ```
2. Abre el puerto mostrado por la plataforma (por defecto `8000`).
3. Verás la app en el navegador integrado de la plataforma.

## Verla localmente (doble clic)

También puedes abrir `index.html` directamente en cualquier navegador.

No requiere dependencias externas.
